Étape 3 : Configuration domaine Infomaniak

1. Connexion Infomaniak

Connectez-vous à votre Manager Infomaniak

2. Gestion DNS

  • Allez dans Domaines → pariskungfu.com
  • Cliquez sur Gestion DNS

3. Configuration enregistrements DNS

# Supprimez les anciens enregistrements A et CNAME
# Ajoutez ces nouveaux enregistrements :

Type: CNAME
Nom: www
Valeur: [votre-site].netlify.app

Type: A
Nom: @
Valeur: 75.2.60.5

# Enregistrement A alternatif Netlify :
Type: A
Nom: @
Valeur: 104.198.14.52

4. Configuration Netlify

  • Dans Netlify → Site Settings → Domain Management
  • Ajoutez pariskungfu.com et www.pariskungfu.com
  • Activez Force HTTPS

Étape 4 : Tests et vérifications

Tests redirections

  • • pariskungfu.com/#histoire → pariskungfu.com/histoire-shaolin.html
  • • pariskungfu.com/#biographie → pariskungfu.com/biographie.html
  • • pariskungfu.com/#contact → pariskungfu.com/contact.html

Outils de vérification

  • • Test redirections : curl -I
  • • Google Console : Soumettre sitemap
  • • PageSpeed Insights : Performance
